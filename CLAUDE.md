# VeganEats China — MuShanghai 活动专属素食工具

## 项目概览
为 2026年5月10日–6月6日上海虹桥阿里中心 muShanghai 活动的非中文素食外籍参与者服务的公益手机 Web 工具。

## 不可变更约束
- 产品形态：纯手机响应式网页（Chrome/Safari 优先），不做小程序/PC端
- 目标用户：仅 muShanghai 活动外籍非中文素食参与者
- 覆盖范围：虹桥阿里中心步行15分钟（1公里）半径内餐厅
- 核心功能：找餐厅决策 + 到店点菜辅助
- 技术栈：Vercel（前端）+ Convex（数据库）+ 高德地图 AMAP（唯一）
- 上线死线：2026-05-23 24:00
- 语言：仅英文界面
- 无登录、无用户数据收集（仅匿名访问统计）

## 技术栈
- Frontend: Next.js / React，Vercel 部署
- Backend: Convex（免费版，每日5万次查询）
- 地图：高德地图 JS API（海外版）
- 性能目标：首屏 ≤1.5s，LCP ≤1.2s
